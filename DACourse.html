Programme Overview

This intensive one-on-one data analysis training programme is designed to prepare participants for the competitive data analysis job market, with a focus on securing remote positions with companies like Turing. The programme covers both industry-standard data analysis tools and academic research analysis techniques.

Curriculum

Module 1: Excel for Data Analysis

  • Data cleaning and preparation techniques
  • Advanced formulas and functions
  • Pivot tables and data visualization
  • Statistical analysis in Excel
  • Practical applications and case studies

Module 2: SQL Fundamentals

  • Database concepts and SQL syntax
  • Data querying and manipulation
  • Joins and relationships
  • Subqueries and CTEs
  • Performance optimization

Module 3: Power BI

  • Dashboard creation and design principles
  • Data modeling and relationships
  • DAX formulas and measures
  • Interactive visualizations
  • Publishing and sharing reports

Module 4: Python for Data Analysis

  • Python fundamentals for data analysis
  • Working with pandas and NumPy
  • Data visualization with matplotlib and seaborn
  • Statistical analysis and hypothesis testing
  • Machine learning basics

Module 5: Academic Research Analysis

  • SPSS fundamentals and data preparation
  • Statistical tests and analysis techniques
  • EViews for econometric analysis
  • Research methodology and data interpretation
  • Creating research reports and visualizations

Module 6: Personal Branding & Job Readiness

  • Building a data analyst portfolio
  • Resume and LinkedIn optimization
  • Technical interview preparation
  • Remote work best practices
  • Job application strategies for platforms like Turing
